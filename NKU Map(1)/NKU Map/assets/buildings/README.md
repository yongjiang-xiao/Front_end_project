# 建筑图片使用说明

## 概述
现在南开大学交互地图支持在点击建筑热点时显示该建筑的图片，以提供更直观的视觉介绍。同时支持图片轮播功能，可以为单个建筑添加多张图片并通过左右按钮切换查看。

## 图片添加方法

### 方法一：在数据中指定自定义图片路径（单张图片）
在 `script.js` 文件的 `HOTSPOT_DATA` 数组中，为建筑对象添加 `image` 字段：

```javascript
{
  id: 20,
  title: "篮球场",
  description: "室外篮球场地，是同学们进行篮球活动和课余锻炼的常用场所。",
  image: "assets/buildings/basketball-court.jpg",  // 自定义图片路径
  left: 23.87,
  top: 54.0272,
  width: 3.6,
  height: 3.6
}
```

### 方法二：添加多张图片（图片轮播）
在 `script.js` 文件的 `HOTSPOT_DATA` 数组中，为建筑对象添加 `images` 数组字段：

```javascript
{
  id: 33,
  title: "马蹄湖",
  description: "南开校园内著名的景观湖，形状似马蹄，是师生休闲散步的优美场所。",
  images: [
    "assets/buildings/horseshoe-lake-1.jpg",
    "assets/buildings/horseshoe-lake-2.jpg",
    "assets/buildings/horseshoe-lake-3.jpg"
  ],
  left: 24.31,
  top: 41.64,
  width: 8,
  height: 8
}
```

## 图片轮播功能
- 当建筑有多张图片时，会显示左右切换按钮
- 图片右下角会显示当前图片序号和总图片数（如 "2/3"）
- 点击左右按钮可以切换图片
- 第一张图片时左按钮会变暗，最后一张图片时右按钮会变暗
- 按钮具有悬停放大效果，增强交互体验
- 鼠标不在图片区域时，切换按钮会自动隐藏

## 图片要求
- 建议尺寸：宽度不超过400像素，高度自适应
- 格式：推荐使用JPG或PNG格式
- 文件大小：建议控制在200KB以内，以确保加载速度

## 功能说明
- 点击建筑热点时，会在右侧信息面板显示建筑图片和描述
- 按Esc键取消选择时，图片会自动隐藏
- 图片具有悬停放大效果，增强交互体验
- 多图片建筑支持轮播切换，提供更全面的视觉展示

## 注意事项
1. 请确保图片文件已放置在正确的路径中
2. 如果找不到图片，系统会显示破损的图片图标
3. 图片命名请使用英文，避免特殊字符和空格
4. 如果同时设置了 `image` 和 `images` 字段，系统会优先使用 `images` 数组
5. 只有在数据中明确指定了图片路径的建筑才会显示图片
